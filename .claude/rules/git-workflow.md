# Git Workflow Rules

Commit prefix kotelezo:
- `fix:` — bug javitas
- `feat:` — uj funkcio
- `docs:` — csak dokumentacio
- `test:` — csak tesztek
- `chore:` — konfiguracio, tooling
- `refactor:` — viselkedes valtozas nelkuli atiras

Commit uzenetbe kerul: mit, miert, teszt szam (pl. "feat: EWMA smoothing BC17 — 870 tests")
Push csak Tamas jovahagyasaval — CC commitol, Tamas pusholja.
